# Business/Product Doc

## Product Vision
Create a simple web app that helps users find potentially undervalued stocks using a transparent scoring model instead of raw financial tables.

## Target User
- Retail investors who want easier stock screening
- Users who care about value, quality, growth, and sentiment together
- Users who want a simple watchlist and clear explanations

## Core Value Proposition
- Custom score instead of generic stock screener filters
- Simple explanation of why a stock is attractive or risky
- Focus on top-500 companies to reduce noise and improve trust

## Core Features
1. Search for companies by name or ticker
2. Add companies to favorites/watchlist
3. View a stock detail page with score breakdown
4. Browse "great stocks to buy" from the top-500 universe only
5. Read short, explainable reasons behind the score

## Non-Goals for MVP
- Real-time trading
- Portfolio execution
- Advanced charting platform
- Social/community investing features

## Success Metrics
- Search-to-analysis conversion rate
- Number of favorites added per active user
- Repeat weekly usage
- Click-through rate from top stocks list to detail pages
- Watchlist retention over 30 days

## Risks
- Users may distrust opaque scoring if explanations are weak
- Data provider costs may grow quickly without caching
- Poor sector normalization can create misleading recommendations

## MVP Release Scope
- Search
- Favorites
- Top-500 ranking list
- One stock detail view
- Rule-based explanation engine

## Release Messaging
This app helps users quickly identify potentially undervalued large-cap stocks with a transparent, multi-factor score. It is designed for research support, not financial advice.
